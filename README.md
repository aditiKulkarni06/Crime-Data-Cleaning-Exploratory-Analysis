## Overview

This project focuses on cleaning, exploring, and analyzing crime data from 2020 to the present to uncover trends, seasonal patterns, regional differences, and potential socioeconomic correlations. 

The project uses Python to transform raw crime records into meaningful insights that can support data-driven decision-making.

---

## Objectives

* Clean and preprocess large-scale crime datasets
* Identify long-term and seasonal crime trends
* Analyze crime distribution across regions
* Examine relationships between crime and unemployment
* Explore demographic patterns in crime victimization
* Forecast future crime trends using time-series modeling

---

## Dataset

* **Primary Dataset:** *Crime Data from 2020 to Present* (Data.gov)

  * Includes crime date, type, location, region, victim demographics, and weapon codes
* **Supplementary Dataset:** Synthetic unemployment data (CSV)

  * Used to explore correlations between unemployment rates and crime trends
